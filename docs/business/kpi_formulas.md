# KPI Формулы — Бизнес-логика

> Этот файл — минимальный контекст для работы с KPI Engine.
> Читать вместе с [apps/kpi.md](../apps/kpi.md).

## Общее

- **6 KPI**, итого **100 баллов**
- **20 ДГД** (регионов) + КГД (итого по РК, не ранжируется)
- Период: накопленный с 01.01 по дату отчёта, или произвольный
- Источники: ИНИС (до 09.07.2025) и ИСНА (с 09.07.2025)
- Глобальный фильтр для KPI 1–3: `management = 'УНА'`

---

## KPI 1 — Доначисление (макс. **10 баллов**)

```
Факт     = SUM(amount_assessed)
           WHERE management='УНА'
             AND completed_date IN [date_from, date_to]

KPI_план = kbk_share_pct[dgd] / 100
           × SUM(планов всех ДГД прошлого года)
           × 1.20

           kbk_share_pct — ручной ввод оператором 1 раз в год

Исполнение = Факт / KPI_план
```

| Исполнение | Баллы |
|-----------|-------|
| ≥ 100% | **10** |
| ≥ 90% | 5 |
| < 90% | 0 |

## KPI 2 — Взыскание (макс. **40 баллов**)

```
Факт     = SUM(amount_collected)
           WHERE management='УНА'
             AND is_accepted=True
             AND completed_date IN [date_from, date_to]

KPI_план = kbk_share_pct[dgd] / 100
           × SUM(планов взысканий всех ДГД прошлого года)
           × 1.20

Исполнение = Факт / KPI_план
```

| Исполнение | Баллы |
|-----------|-------|
| ≥ 100% | **40** |
| ≥ 90% | 20 |
| ≥ 80% | 10 |
| < 80% | 0 |

---

## KPI 3 — Среднее доначисление на 1 проверку (макс. **10 баллов**)

```
Сумма    = SUM(amount_assessed)
           WHERE management='УНА'
             AND is_counted=True
             AND form_type != 'ДФНО'    ← ДФНО исключается!

Кол-во   = COUNT(inspections)
           WHERE management='УНА'
             AND is_counted=True
             AND form_type != 'ДФНО'

Среднее_факт = Сумма / Кол-во

KPI_план = среднее по всем 20 ДГД за прошлый год × 1.20
           ЕДИНЫЙ порог для всех ДГД (не индивидуальный)

Исполнение = Среднее_факт / KPI_план
```

| Исполнение | Баллы |
|-----------|-------|
| ≥ 100% | **10** |
| ≥ 90% | 5 |
| < 90% | **0** ← 80–89% тоже даёт 0! |

> ⚠️ **Важно:** диапазон 80–89% даёт 0 баллов (не 5). Ранее давал, сейчас нет.

## KPI 4 — Коэффициент занятости (макс. **15 баллов**)

```
Проверок    = COUNT(completed_inspections)
              WHERE management='УНА' AND is_counted=True

Сотрудников = ManualInput.staff_count[dgd, year]
              ← ручной ввод оператором 1 раз в год

Месяцев     = кол-во полных месяцев в периоде (для годового = 12)

Коэф        = Проверок / Сотрудников / Месяцев
```

| Коэффициент | Баллы |
|------------|-------|
| ≥ 0.5 | **15** |
| ≥ 0.4 | 5 |
| < 0.4 | 0 |

---

## KPI 5 — Доля проверок > 6 месяцев (макс. **10 баллов**)

```
Все_проводимые = COUNT(ActiveInspection)
                 WHERE is_counted=True
                 ИСКЛЮЧИТЬ: уголовные дела
                 ИСКЛЮЧИТЬ: запросы правоохранительных органов

Долгие         = COUNT WHERE
                 (report_date - prescription_date).days > 180

Доля = Долгие / Все_проводимые × 100%
```

| Доля | Баллы |
|------|-------|
| < 20% | **10** |
| ≥ 20% | 0 |

---

## KPI 6 — Удельный вес отменённых сумм (макс. **15 баллов**)

```
Источник   = отдельная таблица обжалований (ИНИС)
             WHERE is_counted=True
             ИСКЛЮЧИТЬ: акты завершённые > 2 лет до решения Апелл. комиссии
             ИСКЛЮЧИТЬ: акты управлений кроме УНА

Отменено   = SUM(AppealDecision.amount_cancelled)
Доначислено = Факт из KPI 1 (для того же ДГД и периода)

Доля = Отменено / Доначислено × 100%
```

| Доля | Баллы |
|------|-------|
| ≤ 1% | **15** |
| ≤ 2% | 5 |
| > 2% | 0 |

## Итоговый рейтинг

```
Баллы_итого[dgd] = KPI1 + KPI2 + KPI3 + KPI4 + KPI5 + KPI6  (макс. 100)
Позиция[dgd]     = RANK(Баллы_итого, DESC) среди 20 ДГД
КГД (итого)      = rank=None, отображается отдельно
```

## Ручные вводы (ManualInput)

| Параметр | Частота | Кто | KPI |
|----------|---------|-----|-----|
| `kbk_share_pct` — доля ДГД по 4 КБК (%) | 1 раз в год | Оператор | KPI 1, 2 |
| `staff_count` — кол-во сотрудников аудита | 1 раз в год | Оператор | KPI 4 |
